## 核心特性

### 1. 精致的消息显示

- **用户消息**：青色边框，清晰标识
- **AI 回复**：绿色边框，结构化显示
- **系统消息**：灰色分隔线，低调提示
- **错误消息**：红色边框，醒目警告
- **加载状态**：黄色提示，实时反馈

### 2. 智能上下文

自动包含当前仓库信息：
- 📍 当前分支
- 📝 变更文件列表
- 📌 最近提交记录

### 3. 对话历史管理

- 保持最近 6 轮对话上下文
- 自动过滤系统消息和错误
- 支持清空历史重新开始

### 4. 输入历史导航

- ↑/↓ 浏览之前的输入
- 快速重新发送问题
- 自动保存输入记录

## 使用场景

### 场景 1: 快速咨询

**问题**：如何撤销提交？

**操作**：
1. 打开 AI 对话（调用 `ShowAIChat()`）
2. 输入问题
3. 按 Enter 发送
4. 查看 AI 回复

**AI 回复示例**：
```
撤销提交有几种方式：

1️⃣ 保留更改（推荐）
   git reset --soft HEAD~1
   ↳ 提交被撤销，更改保留在暂存区

2️⃣ 保留更改但不暂存
   git reset --mixed HEAD~1
   ↳ 提交被撤销，更改保留在工作区

3️⃣ 完全丢弃（危险）
   git reset --hard HEAD~1
   ↳ 提交和更改都被删除

💡 在 lazygit 中：
   • 在 commits 视图选择提交
   • 按 'g' 打开 reset 菜单
   • 选择 soft/mixed/hard reset
```

### 场景 2: 多轮对话

**第一轮**：
- 你：如何合并分支？
- AI：[解释合并方法]

**第二轮**：
- 你：如果遇到冲突怎么办？
- AI：[解释冲突解决，结合上下文]

**第三轮**：
- 你：能给个具体例子吗？
- AI：[提供详细示例，基于之前的对话]

### 场景 3: 使用预设问题

**操作**：
1. 按 **Ctrl+P** 打开预设问题
2. 选择 "📚 基础操作: 如何撤销最近的提交？"
3. 问题自动填充到输入框
4. 按 Enter 发送

### 场景 4: 复制 AI 回复

**操作**：
1. 在对话视图中按 **Tab** 切换焦点
2. 按 **Ctrl+C** 复制最后一条 AI 回复
3. 粘贴到其他地方使用

### 场景 5: 保存对话记录

**操作**：
1. 按 **Ctrl+S** 保存对话
2. 对话保存为 Markdown 文件
3. 文件名：`ai_chat_YYYYMMDD_HHMMSS.md`

## 代码集成

### 基本使用

```go
// 打开 AI 对话
gui.ShowAIChat()
```

### 从菜单打开

```go
menuItems := []*types.MenuItem{
    {
        Label: "💬 AI 对话",
        OnPress: func() error {
            return gui.ShowAIChat()
        },
        Key: 'a',
    },
}
```

### 从快捷键打开

```go
{
    Key: 'A',
    Handler: gui.ShowAIChat,
    Description: "打开 AI 对话",
}
```

### 检查 AI 是否启用

```go
if gui.c.AI == nil {
    gui.ShowError("AI 未启用", "请先在设置中启用并配置 AI 功能。")
    return nil
}
return gui.ShowAIChat()
```

## 配置要求

### AI 配置

在 `~/.config/lazygit/config.yml` 中配置：

```yaml
ai:
  enabled: true
  activeProfile: "default"
  profiles:
    - name: "default"
      provider: "deepseek"
      apiKey: "your-api-key"
      model: "deepseek-chat"
      maxTokens: 2000
      timeout: 60
```

### 支持的 AI 提供商

- **DeepSeek**：高性价比，推荐
- **OpenAI**：GPT-3.5/GPT-4
- **Anthropic**：Claude
- **Ollama**：本地运行
- **Custom**：自定义 API

## 注意事项

1. **网络连接**
   - 需要稳定的网络连接（除非使用 Ollama）
   - 请求可能需要几秒钟

2. **API 配额**
   - 注意 API 使用配额和费用
   - 建议设置合理的 maxTokens

3. **隐私安全**
   - 对话内容会发送到 AI 服务商
   - 不要包含敏感信息（密码、密钥）
   - 仓库上下文只包含基本信息

4. **对话历史**
   - 历史保存在内存中
   - 关闭窗口后清空
   - 可手动保存到文件
