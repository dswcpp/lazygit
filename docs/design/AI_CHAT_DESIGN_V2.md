# AI 对话功能设计文档

## 设计理念

### 核心原则
1. **终端原生体验**：保持 TUI 风格，使用 Unicode 字符和 ANSI 颜色
2. **流畅交互**：快速响应，异步处理，不阻塞界面
3. **信息密度**：在有限空间内展示最有价值的信息
4. **视觉层次**：清晰的消息分组、角色区分、状态提示
5. **键盘优先**：所有操作都可通过键盘完成

## 功能设计

### 核心功能

1. **多轮对话**
   - 保持对话上下文（最近 10 轮）
   - 自动包含仓库状态
   - 支持话题切换

2. **消息操作**
   - 复制消息内容
   - 复制代码块
   - 重新生成回复
   - 反馈（有用/无用）
   - 保存对话

3. **代码执行**
   - 识别代码块
   - 一键执行命令
   - 执行前确认
   - 显示执行结果

4. **快捷命令**
   - `/help` - 显示帮助
   - `/clear` - 清空对话
   - `/save` - 保存对话
   - `/context` - 显示上下文
   - `/model` - 切换模型

5. **智能建议**
   - 根据当前状态提供建议
   - 常见问题快捷入口
   - 相关命令推荐

### 高级功能

1. **流式输出**
   - 实时显示 AI 回复
   - 逐字显示效果
   - 可中断生成

2. **对话历史**
   - 保存到文件
   - 搜索历史对话
   - 恢复历史会话

3. **多模型支持**
   - 快速切换模型
   - 模型性能对比
   - 自动选择最佳模型

4. **上下文管理**
   - 自动收集相关信息
   - 手动添加上下文
   - 上下文压缩

## 技术实现

### 消息渲染

```go
type MessageRenderer struct {
    width       int
    colorScheme ColorScheme
}

func (r *MessageRenderer) Render(msg ChatMessage) string {
    // 1. 渲染消息头（角色、时间）
    header := r.renderHeader(msg)

    // 2. 解析消息内容（Markdown）
    content := r.parseMarkdown(msg.Content)

    // 3. 渲染代码块
    content = r.renderCodeBlocks(content)

    // 4. 渲染列表
    content = r.renderLists(content)

    // 5. 渲染操作按钮
    actions := r.renderActions(msg)

    // 6. 组合并添加边框
    return r.wrapInBox(header, content, actions)
}
```

### 流式输出

```go
func (chat *AIChat) streamResponse(prompt string) {
    stream := chat.gui.c.AI.Stream(chat.ctx, prompt)

    currentMsg := &ChatMessage{
        Role:      "assistant",
        Content:   "",
        Timestamp: time.Now(),
    }

    for chunk := range stream {
        currentMsg.Content += chunk
        chat.gui.g.Update(func(*gocui.Gui) error {
            chat.renderMessage(currentMsg)
            return nil
        })
    }
}
```

### 代码执行

```go
func (chat *AIChat) executeCode(code string) error {
    // 1. 显示确认对话框
    confirmed := chat.gui.ShowConfirm(
        "执行命令",
        fmt.Sprintf("确定要执行以下命令吗？\n\n%s", code),
    )

    if !confirmed {
        return nil
    }

    // 2. 执行命令
    result, err := chat.gui.c.OS().Cmd.NewShell(code).RunWithOutput()

    // 3. 显示结果
    chat.addSystemMessage(fmt.Sprintf(
        "执行结果:\n%s", result,
    ))

    return err
}
```

## 性能优化

1. **消息缓存**
   - 缓存渲染结果
   - 只重新渲染变化的部分

2. **虚拟滚动**
   - 只渲染可见区域
   - 大量消息时保持流畅

3. **异步处理**
   - AI 调用异步
   - 渲染异步
   - 不阻塞主线程

4. **内存管理**
   - 限制历史消息数量
   - 自动清理旧消息
   - 压缩长消息

## 错误处理

1. **网络错误**

3. **输入验证**
