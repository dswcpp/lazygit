# 范围选择

某些操作可以在一系列连续的项目上执行。例如：
* 一次暂存多个文件
* 一次压缩多个提交
* 一次复制（用于 cherry-pick）多个提交

有两种方法可以选择一系列项目：
1. 粘性范围选择：按 'v' 切换范围选择，然后使用上/下箭头键扩展选择。要重置选择，再次按 'v'。
2. 非粘性范围选择：按 shift+上 或 shift+下 扩展选择。要重置选择，按上/下而不按 shift。

粘性选项对 vim 用户来说会更熟悉，第二个选项对不习惯以模态方式做事的用户来说会感觉更自然。

要对一系列项目执行操作，只需按该操作的正常键即可。如果该操作仅适用于单个项目，它将引发错误。这是一个新功能，计划是逐步支持越来越多的操作的范围选择。如果有一个你希望支持范围选择但目前不支持的操作，请在仓库中提出问题。
