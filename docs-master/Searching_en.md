# 搜索/过滤

## 视图搜索/过滤

根据当前聚焦的视图，按 '/' 将弹出过滤或搜索提示。过滤时，视图的内容将被过滤为仅显示与查询字符串匹配的行。搜索时，视图的内容不会被过滤，但匹配的行会被高亮显示，你可以使用 `n`/`N` 在匹配项之间迭代。

我们打算很快为文件视图支持过滤，但目前它使用搜索。我们打算继续为提交视图使用搜索，因为你通常关心匹配提交之前/之后的提交。

如果你希望在给定视图上同时启用过滤和搜索，请为此提出问题。

## 按状态过滤文件

你可以通过在文件视图中按 `<c-b>` 来过滤文件视图，仅显示已暂存/未暂存的文件。

## 按文件路径过滤提交

你可以过滤提交视图，仅显示包含对给定文件路径更改的提交。

你可以通过以下几种方式执行此操作：
1) 使用 -f 标志启动 lazygit，例如 `lazygit -f my/path`
2) 在 lazygit 中，按 `<c-s>`，然后输入要过滤的文件路径
