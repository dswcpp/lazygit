# MessageBox 设计改进方案

## 设计理念

保持终端原生 TUI 风格，使用 ASCII 字符和 Box Drawing Characters，提升视觉层次感和可读性。

## 改进方向

### 1. 视觉层次优化

#### 当前问题
- 消息框边框单一，缺乏层次感
- 标题和内容区分不明显
- 按钮提示不够突出

#### 改进方案

**双线边框设计**（使用 Box Drawing Characters）：
```
╔═══════════════════════════════════════╗
║ ⚠ 确认删除                            ║
╟───────────────────────────────────────╢
║                                       ║
║  确定要删除分支 'feature-x' 吗？      ║
║                                       ║
║  此操作不可撤销。                     ║
║                                       ║
╟───────────────────────────────────────╢
║  [Enter] 确认  [Esc] 取消             ║
╚═══════════════════════════════════════╝
```

**单线边框设计**（简洁风格）：
```
┌───────────────────────────────────────┐
│ ⚠ 确认删除                            │
├───────────────────────────────────────┤
│                                       │
│  确定要删除分支 'feature-x' 吗？      │
│                                       │
│  此操作不可撤销。                     │
│                                       │
├───────────────────────────────────────┤
│  [Enter] 确认  [Esc] 取消             │
└───────────────────────────────────────┘
```

**圆角边框设计**（现代风格）：
```
╭───────────────────────────────────────╮
│ ⚠ 确认删除                            │
├───────────────────────────────────────┤
│                                       │
│  确定要删除分支 'feature-x' 吗？      │
│                                       │
│  此操作不可撤销。                     │
│                                       │
├───────────────────────────────────────┤
│  [Enter] 确认  [Esc] 取消             │
╰───────────────────────────────────────╯
```

### 2. 消息类型图标

使用 Unicode 字符表示不同消息类型：

| 类型 | 图标 | 颜色 | 用途 |
|------|------|------|------|
| 信息 | ℹ | 蓝色 | 一般信息提示 |
| 成功 | ✓ | 绿色 | 操作成功 |
| 警告 | ⚠ | 黄色 | 需要注意 |
| 错误 | ✗ | 红色 | 错误信息 |
| 问题 | ? | 青色 | 需要确认 |

### 3. Toast 通知优化

#### 当前 Toast 显示方式
- 显示在底部状态栏
- 持续时间固定
- 无视觉层次

#### 改进方案

**位置选项**：
- 右上角（推荐）：不遮挡主要内容
- 右下角：传统位置
- 顶部中央：重要通知

**Toast 样式**：
```
╭─────────────────────────────╮
│ ✓ 分支已成功删除             │
╰─────────────────────────────╯
```

**多 Toast 堆叠**：
```
╭─────────────────────────────╮
│ ✓ 文件已保存                 │
╰─────────────────────────────╯
╭─────────────────────────────╮
│ ℹ 正在推送到远程仓库...      │
╰─────────────────────────────╯
```

### 4. 按钮样式改进

#### 当前样式
```
[Enter] 确认  [Esc] 取消
```

#### 改进样式

**高亮当前选项**：
```
┌─────────┐  ┌─────────┐
│ ✓ 确认  │  │   取消  │
└─────────┘  └─────────┘
   (Enter)      (Esc)
```

**简洁样式**：
```
[ ✓ 确认 ]  [ ✗ 取消 ]
  (Enter)     (Esc)
```

### 5. 进度指示器

**加载中**：
```
╭───────────────────────────────────────╮
│ ⏳ 正在处理...                        │
│                                       │
│  [████████░░░░░░░░░░] 40%            │
│                                       │
│  正在推送到远程仓库...                │
╰───────────────────────────────────────╯
```

**无限加载**：
```
╭───────────────────────────────────────╮

│                                       │
│  ⠋ 正在连接服务器...                 │
```

## 配置选项

在 `config.yml` 中添加配置：

```yaml
gui:
  messageBox:
    # 边框样式: "single", "double", "rounded", "bold"
    borderStyle: "rounded"

    # Toast 位置: "top-right", "top-center", "bottom-right"
    toastPosition: "top-right"

    # Toast 持续时间（秒）
    toastDuration: 3

    # 是否显示图标
    showIcons: true

    # 是否使用颜色
    useColors: true
```

## 实现优先级

### Phase 1: 基础改进（必须）
1. ✅ 改进边框样式（支持多种边框类型）
2. ✅ 添加消息类型图标
3. ✅ 优化标题和内容区分

### Phase 2: Toast 优化（推荐）
1. ✅ Toast 位置可配置
2. ✅ Toast 堆叠显示
3. ✅ Toast 自动消失动画

### Phase 3: 高级功能（可选）
1. ⬜ 进度指示器
2. ⬜ 多按钮支持
3. ⬜ 输入框样式优化

## 兼容性考虑

- 所有字符使用 UTF-8 编码
- 支持降级到 ASCII 字符（当终端不支持 Unicode 时）
- 颜色支持检测（256 色 / 16 色 / 无色）
- 保持与现有 API 的兼容性

### 消息类型定义

```go
type MessageType int

const (
    MessageTypeInfo MessageType = iota
    MessageTypeSuccess
    MessageTypeWarning
    MessageTypeError
    MessageTypeQuestion
)

func (mt MessageType) Icon() string {
    icons := map[MessageType]string{
        MessageTypeInfo:     "ℹ",
        MessageTypeSuccess:  "✓",
        MessageTypeWarning:  "⚠",
        MessageTypeError:    "✗",
        MessageTypeQuestion: "?",
    }
    return icons[mt]
}

func (mt MessageType) Color() style.TextStyle {
    colors := map[MessageType]style.TextStyle{
        MessageTypeInfo:     style.FgCyan,
        MessageTypeSuccess:  style.FgGreen,
        MessageTypeWarning:  style.FgYellow,
        MessageTypeError:    style.FgRed,
        MessageTypeQuestion: style.FgCyan,
    }
    return colors[mt]
}
```
